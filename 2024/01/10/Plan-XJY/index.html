2024年2月计划表

最后一周计划

2月26 : 保证流程正确

  • 完成 cocotb-axi的 Testbench 流程搭建，具体如下：
    • 使用 cocotb-axi repo 下的 function write/read ，使用 axi-lite Master 对 axi-lite interface axi-RAM 读写 ；
    • 使用 cocotb-axi repo 下的 function write/read ，使用 axi-full Master 对 axi-full interface axi-RAM 读写 ；
    • 其他要求：
        1. 暂时不需要学会使用 axi-test 的使用方式，暂时学会 basic 即可；
        2. 学会：1）写cocotb-test.py测试文件 ，包括对相关 interface RAM 的调用和 write/read function 的使用；2）学会相关 Makefile 文件的基本使用，要将 .vvd-> .vcd 生成，可以使用gtkwave 查看相应的读写波形；
